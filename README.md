## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/harkiratsm/food-server.git
```

2. Change the working directory

```bash
cd food-server
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run start
```

🚀 You are all set to go
